This document explains how to set up the LinkedGeoData RDF-Delta Osmosis-Plugin.
Document as of 24. Aug. 2010, Revision 136. 

Overview of the tasks need to be performed:
1. Download required libraries
2. Copy the plugin to a location where osmosis can find it
3. Add the required libraries AND the plugin to the class path

These steps are explained in the following.


1. Download required libraries
-------------------------------------------------------------------------------
The requried libraries are those specified in the pom.xml of the LinkedGeoData

2. Copy the plugin to a location where osmosis can find it
-------------------------------------------------------------------------------
According to [1], in order to install an osmosis plugin, copy it to one of the
following locations:

To install these tasks, copy the specified zip-file into

    * ~/.openstreetmap/osmosis/plugins (Linux) or
    * "C:\\Documents and Settings\\(Username)\\Application Data\\Openstreetmap\\Osmosis\\Plugins" (english Windows) or
    * "C:\\Dokumente und Einstellungen\\(Username)\\Anwendungsdaten\\Openstreetmap\\Osmosis\\Plugins" (german Windows) or
    * the current directoy or
    * the subdirectory plugins in the current directory 


On linux machines we recommend the first option, therefore locating the plugin
at
~/.openstreetmap/osmosis/plugins/RDFDeltaOSMPlugin.zip


[1] http://wiki.openstreetmap.org/wiki/Osmosis/Detailed_Usage#Plugin_Tasks


3. Add the required libraries AND the plugin to the class path
-------------------------------------------------------------------------------
Assuming all libraries have been copied into <osmosis-root>/lib/lgd,
edit the file <osmosis-root>/config/plexus.inf and add the following lines
to the [osmosis.core] section:

load ${app.home}/lib/lgd/slf4j-jdk14-1.6.0.jar

load ${app.home}/lib/lgd/*.jar
load ~/.openstreetmap/osmosis/plugins/RDFDeltaOSMPlugin.zip
